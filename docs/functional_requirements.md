# FileSyncMonitor 软件功能需求说明书 (SRS)

## 1. 软件概述
FileSyncMonitor 是一款专为 macOS 设计的高性能目录监控与同步管理工具。它能够实时捕获本地文件系统的任何细微变动，并提供直观的界面供用户查看、分析以及同步至腾讯 IMA 云端知识库。

## 2. 核心功能需求

### 2.1 实时文件监控 (File Monitoring)
- **底层驱动**：基于 macOS 原生 `FSEvents` 框架，实现对指定目录的内核级监控。
- **改动捕获**：支持捕获文件的“创建 (Create)”、“修改 (Modify)”、“删除 (Delete)”及“重命名 (Rename)”事件。
- **去抖合并**：内置 2.0 秒的事件合并算法，在高频操作下极大地降低 CPU 占用。
- **权限持久化**：采用 `Security-Scoped Bookmarks` 技术，确保应用重启后依然拥有对监控目录的访问权限。

### 2.2 数据持久化与查询 (Storage)
- **存储引擎**：使用 `SwiftData` (基于 SQLite) 进行本地存储。
- **多维查询**：支持按时间倒序查看所有改动记录，并支持按文件名、路径或同步状态进行过滤。

### 2.3 腾讯 IMA 云同步 (Cloud Sync)
- **API 集成**：对接腾讯 IMA OpenAPI。
- **一键上传**：在界面中提供一键同步按钮，将检测到的文档变动实时导入 IMA 知识库。
- **状态追踪**：自动记录每个文件的同步状态（已同步/待同步）。

### 2.4 报告与导出 (Reporting & Export)
- **统计看板**：提供今日、本周、本月的改动趋势分析卡片。
- **多格式导出**：支持将监控到的历史记录导出为 `CSV` 或 `JSON` 格式。

### 2.5 系统级集成 (System Integration)
- **菜单栏管理**：支持常驻系统菜单栏，动态显示未同步的文件数量角标。
- **本地通知**：在后台监控到改动时，通过 macOS 系统通知中心即时提醒用户。

## 3. UI/UX 设计规范 (IMA Aesthetic)

### 3.1 视觉风格
- **品牌色**：腾讯蓝 (#0052D9)。
- **材质**：侧边栏使用超薄毛玻璃效果 (`.ultraThinMaterial`)。
- **布局**：采用 `NavigationSplitView` 侧边栏结构，遵循 macOS 现代应用交互逻辑。

### 3.2 交互反馈
- **微动效**：按钮点击、视图切换均带有平滑的弹簧动画 (`.spring`)。
- **多状态显示**：提供列表视图 (List) 和层级树视图 (Tree) 两种展示模式。

## 4. 性能与安全性
- **性能目标**：静默期 CPU 占用 < 0.1%，内存占用 < 150MB。
- **安全性**：API 密钥等敏感信息建议后续存储至系统 Keychain。

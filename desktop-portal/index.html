关于 FileSyncMonitor

FileSyncMonitor 是一款面向 macOS 用户的轻量化文件夹自动监控与同步工具。

它能够静默监控您指定的本地文件夹，捕获任何新建、修改、删除和重命名等文件变动事件，并能够安全地自动或手动同步到腾讯 IMA 个人知识库。在这里，我们将帮助您在本地常用编辑器与云端平台之间，架起一座可靠的双向同步桥梁。

常见问题与解答 (FAQ)

解决首次配置、微信登录失效、删除同步限制与本地文件安全等常见疑惑。

首次打开应用，主界面会显示微信登录二维码。请使用微信扫码授权登录以绑定您的腾讯 IMA 账号。登录成功后，前往「设置 -> 监控目录」点击添加您想监控的文件夹，并为该文件夹绑定对应的云端知识库，即可开始监控和同步文件。

本应用通过内嵌的腾讯官方安全页面进行登录，所有登录凭据（Token/Cookie）均在本地沙盒内安全加密存储，不会泄露。如果因为长时间未运行或腾讯侧策略调整导致授权失效，应用会在同步时提示“授权已过期”，此时只需在主界面重新扫码登录即可恢复。

为了防止编辑器保存临时状态时产生大量高频碎片记录，应用引入了 2 秒的防抖延迟机制（Debounce）。当您对文件进行连续编辑或保存时，应用会等待文件写入稳定后统一生成一条记录。另外，被忽略规则过滤的文件也不会进入队列。

绝对不会。无论是在设置中点击「清除文件记录」还是「彻底重置应用」，都只属于应用自身数据的管理操作（清空本地 SQLite 数据库中的事件历史、断开文件夹监控或清除偏好设置），绝对不会以任何形式修改或删除您电脑磁盘上的任何物理文件，请放心操作。

忽略规则用于在事件产生前进行过滤，避免日志杂乱。您可以在「设置 -> 忽略规则」中通过以下三种方式过滤：
1. 【全名过滤】：输入完整文件名（如 config.json），完全匹配的文件会被过滤。
2. 【后缀过滤】：输入以点开头的扩展名（如 .tmp 或 .log），匹配该类型的所有临时文件。
3. 【目录过滤】：输入文件夹名称（如 node_modules 或 build），匹配该目录下的所有变动均被屏蔽。

本地到云端同步逻辑

了解本地目录中的变更（增、删、改、重命名）如何处理并推送到腾讯 IMA 个人知识库。

01

变更捕获与防抖

基于 OS 级别的底层监视（macOS FSEvents），程序会静默捕获目录内所有文件的读写变化。为防止文本保存时产生高频碎片的冗余写入，应用引入了 2 秒防抖机制，待文件写入稳定后，才生成一条合规记录进入队列。

02

本地事件抵消与合并

【新建并删除抵消】如果在文件同步到云端前，本地发生了删除，应用会自动检测并直接抹除数据库中对应的未同步创建记录，两起事件完全抵消，不产生任何同步任务与冗余记录。对于频繁发生的修改，只会合并更新时间戳，绝不产生重复记录。

03

云端增量更新

【同步避环】同步开始时会暂停本地文件监视，同步完毕后重新启用，彻底杜绝同步下载本身触发的“本地二次变动”死循环。【增量执行】对于新建及修改事件，上传文件并更新保存云端关联 ID；对于删除事件，若检测到该文件曾有云端绑定历史，则调用腾讯 IMA 接口对云端对应文档一并进行物理删除。

监控与过滤

从选择目录到过滤噪声文件，保证记录列表干净可用。

添加监控目录

在首页或设置中添加文件夹，应用会保存授权并在下次启动后继续监控。

实时捕获变动

通过 macOS FSEvents 捕获新增、修改、删除和重命名，变动会自动进入记录列表。

默认忽略规则

默认过滤 .DS_Store、临时文件、系统目录、构建产物和常见缓存目录。

自定义忽略项

可在设置中维护忽略文件名、后缀和目录名，让监控范围更贴合你的项目。

记录处理

围绕待同步和全部记录进行查看、筛选、确认与清理。

待同步队列

左侧第二个入口集中展示未处理变动，可搜索路径、按类型筛选并批量处理。

全部记录视图

保留所有已同步和未同步记录，适合回溯历史、查找文件变动轨迹。

列表与树状模式

二级侧栏支持列表和树状视图切换，既能快速扫列表，也能按目录结构定位。

详情与快捷操作

选中记录后可查看路径、时间、状态，并执行同步、标记完成、Finder 显示或删除记录。

批量删除记录

在待同步或全部记录页可清理当前筛选结果，只删除应用内记录，不影响真实文件。

IMA 云端同步

支持手动上传、自动上传、知识库绑定和云端拉取。

手动同步

默认采用手动模式，点击单条“同步到 IMA”或首页“全部同步”后再上传。

自动同步

开启后，文件稳定 2 秒会自动上传，减少频繁保存造成的重复同步。

目录绑定知识库

每个监控目录都可以绑定不同 IMA 知识库；不绑定时默认导入为新笔记。

从云端拉取更新

待同步页可从 IMA 拉取云端新增内容，并下载到对应本地目录。

删除事件同步提示

由于 IMA 未开放云端删除 API，本地删除会在应用内标记已同步，云端文档需手动清理。

请求日志排查

连接测试和同步请求会记录在 IMA 请求日志中，方便排查凭据、接口和网络问题。

腾讯 IMA 账户登录

使用微信扫码授权登录，快速绑定知识库并开启云端同步。

微信扫码授权

首次打开应用或在未登录状态下，主界面会显示微信登录二维码，扫码即可安全登录。

管理绑定设备

在“设置 > 同步”中可查看当前账号已登录的其他多端设备，并可随时安全退出登录。

查看存储容量

登录后，在“设置 > 同步”中可直观查看云端个人空间的使用情况和总配额。

同步至知识库

登录成功后即可拉取云端知识库列表，在“设置 > 监控目录”中可以为不同监控目录绑定具体知识库。

报告、通知与菜单栏

把同步状态从主窗口延伸到菜单栏、通知和导出文件。

统计报告

报告页按今天、近 7 天、近 30 天或全部范围统计记录、待同步和已完成数量。

CSV / JSON 导出

可导出当前记录或报告范围，便于留档、分析或给其他工具继续处理。

系统通知提醒

新文件变动出现时可发送 macOS 通知，避免待同步事项被遗忘。

菜单栏状态

菜单栏图标会显示待同步数量，可快速打开窗口、标记完成或退出应用。

设置与体验

让界面语言、外观以及开机自启等偏好设置保持可控。

多语言界面

支持跟随系统、English和简体中文，切换后主界面和菜单栏同步刷新。

浅色与深色模式

可跟随系统，也可手动选择浅色或深色，整体保持清新简约风格。

开机自动运行

可在“设置 > 通知与导出”中开启开机自动运行，使本应用在后台静默为您守护文件。
